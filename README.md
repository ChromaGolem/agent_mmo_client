# Agent MMO

A multiplayer dungeon crawler played through Claude Code. You are a demigod. Your hero is your champion in the Underworld. Scheme together, plan expeditions, explore dungeons — all through a terminal conversation.

## Requirements

- [Claude Code](https://claude.ai/code) with a Claude subscription
- Node.js 22+

## Quick Start

```bash
git clone <this-repo>
cd agent_mmo_client
./play.sh
```

First run prompts for a username and password (saved locally in `.env`). Your hero greets you and you're in.

## What You Can Do

- **Explore** a procedurally generated dungeon that's different every delve
- **Fight** monsters with dice-based combat — the deeper you go, the harder they get
- **Level up** and allocate stats (Strength, Vitality, Agility, Intelligence)
- **Buy gear** at the town shops — 10 weapon tiers, 8 armor tiers
- **Meet other heroes** — chat, trade, and fight together in parties
- **Face 50 named bosses** every 5th floor with unique abilities and guaranteed rare loot

## Multiplayer

Two players in the same location can:
- Talk
- Form a party
- Fight together
